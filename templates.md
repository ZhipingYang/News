# 资讯类型模板定义

本文档定义了不同资讯类型的模板结构和侧重点，用于 MCP 工具自动生成标准化的资讯内容。

## 模板设计原则

- **结构化**：统一的板块划分，便于阅读和对比
- **差异化**：根据资讯类型调整侧重点
- **专业性**：面向企业和专业读者
- **可读性**：使用 emoji、分点、符号增强视觉效果

## 使用指南

### 如何选择模板

MCP 工具会根据以下规则自动选择模板：

1. **关键词匹配**：
   - AI编程：coding, programming, developer, IDE, Copilot, code generation
   - AI芯片：chip, GPU, TPU, NPU, processor, semiconductor
   - 量子计算：quantum, qubit, superposition, entanglement
   - 机器人：robot, robotics, automation, manipulator
   - 生成式AI：GPT, LLM, diffusion, generation, ChatGPT, Midjourney

2. **内容分析**：
   - 技术深度：代码示例 → AI编程
   - 硬件指标：制程、算力 → AI芯片
   - 科学原理：量子态、纠错 → 量子计算
   - 物理系统：运动、感知 → 机器人
   - 创作能力：文本、图像生成 → 生成式AI

### 模板定制说明

每个模板的必填板块：
- ✅ 新闻背景
- ✅ 技术现状/技术突破
- ✅ 行业应用现状/应用前景
- ✅ 结论

可选板块（根据信息完整度）：
- 市场与商业影响
- 战略与未来展望
- 代码示例（AI编程专属）
- 技术对比表格

### 质量标准

生成的资讯内容应满足：
1. **完整性**：包含所有必填板块
2. **准确性**：数据来源可靠，描述准确
3. **可读性**：使用 emoji、分点、符号增强视觉
4. **专业性**：面向企业和专业读者，避免过度简化
5. **时效性**：信息在 7 天内
